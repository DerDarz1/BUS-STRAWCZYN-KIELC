# Rozkład Busów 🚌

Rozkład jazdy busów na trasie **Ruda – Kielce** (KOWMAR zwykły, KOWMAR szybki, MAJCHRAK).

🌐 **Strona:** [https://TWOJA-NAZWA.github.io/rozklad-busow](https://TWOJA-NAZWA.github.io/rozklad-busow)

## Funkcje

- Najbliższy kurs **do Kielc** i **z Kielc** — live, co sekundę
- Przełącznik przewoźnika: KOWMAR zwykły / KOWMAR szybki / MAJCHRAK
- Przełącznik kierunku
- Przełącznik dnia: Pon–Pt / Sobota
- Animacja busa przy zmianie zakładki 🚌
- Tryb ciemny / jasny

## Uruchomienie lokalnie

Otwórz `index.html` w przeglądarce — nie wymaga serwera ani instalacji.

## Wdrożenie na GitHub Pages

1. Wgraj repozytorium na GitHub
2. Wejdź w **Settings → Pages**
3. Source: `Deploy from a branch` → branch `main` → folder `/ (root)`
4. Zapisz — strona będzie dostępna pod `https://TWOJA-NAZWA.github.io/NAZWA-REPO`
